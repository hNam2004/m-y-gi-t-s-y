# IoT System for Turning Machine

## Project Structure
This repository contains the following folders:

- **requirement/** - Contains project requirements and specifications.
- **docs/** - Stores schematics and datasheets of sensors related to the project.
- **design/** - System design, diagrams, and architecture.
- **code/** - Source code for the IoT system.

## Project Overview
This project aims to develop an IoT system for a turning machine. The system will enable real-time monitoring and control, improving efficiency and automation.
